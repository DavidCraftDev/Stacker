# Stacker

A plugin to stack player above player! <br/>
This plugin works with the Minecraft versions 1.11-1.17.1

# Permissions

stacker.use.self <br/>
stacker.use.other <br/>
stacker.immune.other <br/>
stacker.immune.self <br/>
stacker.help
